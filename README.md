# Bonjour, je suis Celio Mozes Rocha !

Développeur **React** et **Android** passionné par le rendu graphique performant, la logique complexe et les applications mobiles natives.


---

## Projets en cours

### Jeu d'Échecs
Développement d'un moteur d'échecs complet avec une interface utilisateur réactive en **React**.
- **Technos :** React, Logique de jeu, UI Interactive
- [Voir le dépôt](https://github.com/celio-mozes-rocha/ChessLite)

### Planetarium
Application de visualisation animée du ciel.
- **Architecture Hybride :** 
  - **Leaflet (OSM)** : Gestion de la carte terrain et géolocalisation (sélection de position via clic ou recherche Nominatim).
  - **HTML5 Canvas** : Rendu efficace des objets célestes (étoiles, constellations, trajectoires) avec une projection graphique personnalisée.
- **Technos :** React, Canvas API, Leaflet, OpenStreetMap, Nominatim API
- [Voir le dépôt](https://github.com/celio-mozes-rocha/planetarium-2D)

### App Météo Android
Application native Android pour consulter les prévisions météorologiques en temps réel.
- **Technos :** Kotlin, Android, API Météo, Material Design
- [Voir le dépôt](https://github.com/celio-mozes-rocha/AppMeteo)
