### ♟️ Jeu d'Échecs
Développement d'un moteur d'échecs complet avec une interface utilisateur réactive en **React**.
- **Technos :** React, Logique de jeu, UI Interactive
- [Voir le dépôt](https://github.com/celio-mozes-rocha/chess-game) *(Lien à mettre à jour)*

### 🌍 Planetarium
Application de visualisation cartographique avancée.
- **Spécificité :** Implémentation d'une **projection graphique personnalisée** via **HTML5 Canvas** pour des performances optimales et un rendu unique.
